## Preguntas de Selección Múltiple: Productos Notables (1° Medio)

**Instrucciones:** Lee atentamente cada pregunta y selecciona la alternativa correcta.

---

**1. El desarrollo de $(x+2)^2$ es:**\
a) $x^2 + 4\
    b) $x^2 + 4x + 4$\
    c) $x^2 + 2x + 4$\
    d) $x^2 + x + 4$

**2. ¿Cuál de las siguientes expresiones representa el desarrollo del cuadrado de binomio $(a-3)^2$?**
    a) $a^2 - 9$
    b) $a^2 - 6a + 9$
    c) $a^2 + 6a + 9$
    d) $a^2 - 3a + 9$

**3. El producto $(y+5)(y-5)$ corresponde a:**
    a) $y^2 - 25$
    b) $y^2 + 25$
    c) $y^2 - 10y + 25$
    d) $y^2 + 10y + 25$

**4. Al desarrollar $(2m+1)^2$, obtenemos:**
    a) $4m^2 + 1$
    b) $4m^2 + 2m + 1$
    c) $4m^2 + 4m + 1$
    d) $4m^2 + m + 1$

**5. La expresión $(3z-2)^2$ se desarrolla como:**
    a) $9z^2 - 4$
    b) $9z^2 - 12z + 4$
    c) $9z^2 + 12z + 4$
    d) $9z^2 - 6z + 4$

**6. El producto $(5p+q)(5p-q)$ es igual a:**
    a) $25p^2 + q^2$
    b) $25p^2 - q^2$
    c) $25p^2 - 10pq + q^2$
    d) $25p^2 + 10pq + q^2$

**7. Al calcular $(x+y+z)^2$, ¿cuál es el resultado correcto?**
    a) $x^2 + y^2 + z^2$
    b) $x^2 + y^2 + z^2 + 2xy + 2xz + 2yz$
    c) $x^2 + y^2 + z^2 + xy + xz + yz$
    d) $x^2 + y^2 + z^2 + 2(xy + xz + yz)$

**8. Si se sabe que $a^2 - b^2 = (a+b)(a-b)$, entonces $100 - 49x^2$ es igual a:**
    a) $(10+7x)(10-7x)$
    b) $(10+7x)(10+7x)$
    c) $(10-7x)(10-7x)$
    d) $(100-7x)(100+7x)$

**9. El desarrollo de $(a^2+b)^2$ es:**
    a) $a^4 + b^2$
    b) $a^4 + 2a^2b + b^2$
    c) $a^4 + ab^2 + b^2$
    d) $a^4 + 2ab + b^2$

**10. Si $(x-k)^2 = x^2 - 14x + 49$, ¿cuál es el valor de $k$?**
    a) 7
    b) -7
    c) 14
    d) -14

**11. La expresión $(2x+3y)(2x-3y)$ se simplifica a:**
    a) $4x^2 + 9y^2$
    b) $4x^2 - 9y^2$
    c) $4x^2 - 12xy + 9y^2$
    d) $4x^2 + 12xy + 9y^2$

**12. ¿Cuál de las siguientes igualdades es INCORRECTA?**
    a) $(a+b)^2 = a^2 + 2ab + b^2$
    b) $(a-b)^2 = a^2 - 2ab + b^2$
    c) $a^2 - b^2 = (a+b)(a-b)$
    d) $(a+b)^2 = a^2 + b^2$

**13. Si el área de un cuadrado es $(x-4)^2$ unidades cuadradas, su lado mide:**
    a) $x^2 - 16$
    b) $x-16$
    c) $x-4$
    d) $x+4$

**14. El producto $(2a+1)(a+3)$ se desarrolla como:**
    a) $2a^2 + 3$
    b) $2a^2 + 7a + 3$
    c) $2a^2 + 5a + 3$
    d) $2a^2 + 6a + 3$

**15. Si $(x+y)(x+y) = 16$ y $(x-y)(x-y) = 4$, ¿cuál es el valor de $xy$?**
    a) 3
    b) 4
    c) 5
    d) 6

**16. El desarrollo de $(m^2-n^2)^2$ es:**
    a) $m^4 - n^4$
    b) $m^4 - 2m^2n^2 + n^4$
    c) $m^4 - m^2n^2 + n^4$
    d) $m^4 + 2m^2n^2 + n^4$

**17. Al simplificar $(3x-1)(3x+1) - (3x)^2$, el resultado es:**
    a) 0
    b) -1
    c) $9x^2 - 1$
    d) $1$

**18. Si $a+b = 7$ y $a-b = 3$, ¿cuál es el valor de $a^2 - b^2$?**
    a) 4
    b) 10
    c) 21
    d) 49

**19. ¿Cuál de las siguientes expresiones es un trinomio cuadrado perfecto?**
    a) $x^2 + 9$
    b) $x^2 - 6x - 9$
    c) $x^2 - 6x + 9$
    d) $x^2 + 6x + 18$

**20. El área de un rectángulo es $4x^2 - 9$. Si uno de sus lados mide $2x+3$, el otro lado mide:**
    a) $2x-3$
    b) $4x^2-9$
    c) $x-3$
    d) $2x+3$
